# Greeting

이 스킬이 호출되면 다음을 수행한다.

1. 사용자에게 한국어(합쇼체)로 따뜻하게 인사한다.
2. `my-cowork-plugin` 마켓플레이스와 `hello-cowork` 플러그인이 정상 설치되어 동작 중임을 확인해 준다.
3. 마켓플레이스의 플러그인이 담을 수 있는 구성 요소(스킬, MCP 서버, 에이전트)를 한 줄로 짧게 안내한다.
4. 직접 플러그인을 추가하고 싶다면 저장소 최상위 `README.md`의 "새 플러그인 추가하기" 절을 참고하라고 알려 준다.

응답은 3~4문장 이내로 짧게 유지한다. 불릿이나 과한 서식은 쓰지 않는다.
