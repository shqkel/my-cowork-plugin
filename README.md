# my-cowork-plugin

Cowork 플러그인을 위한 마켓플레이스 프로젝트다.

## 개요

이 저장소는 Cowork에서 사용할 플러그인과 마켓플레이스 설정을 관리한다.
플러그인과 마켓플레이스 정책에 맞는 파일 구조를 유지하는 것을 목표로 한다.

## 폴더 구조

```
my-cowork-plugin/
├── README.md      # 프로젝트 설명
└── .gitignore     # git 추적에서 제외할 파일 목록
```

(작업이 진행되면서 플러그인 디렉터리와 마켓플레이스 설정 파일이 추가된다.)
